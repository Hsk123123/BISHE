# 基于 Spring Boot + Vue 3 的个人工作室预约服务平台搭建计划

## 项目架构

* **前端**：Vue 3 + TypeScript + Vant UI (移动端) / Element Plus (后台管理)

* **后端**：Spring Boot 3.x + MyBatis Plus

* **数据库**：MySQL 8.0 + Redis

* **文件存储**：MinIO (本地模拟)

## 实施步骤

### 第一阶段：项目初始化与环境搭建

1. **创建项目目录结构**

   * `skill-chain/` (根目录)

   * `skill-chain-backend/` (Spring Boot后端)

   * `skill-chain-frontend/` (Vue 3前端)

2. **后端项目搭建**

   * 初始化Spring Boot 3.x项目（包含Web、MySQL、Redis、MyBatis Plus等依赖）

   * 配置application.yml（数据库、Redis、文件上传等）

   * 创建基础包结构（controller、service、mapper、entity、config、utils等）

   * 配置跨域CORS

   * 配置MyBatis Plus分页插件

3. **前端项目搭建**

   * 初始化Vue 3 + TypeScript + Vite项目

   * 安装并配置Vant UI（移动端）和Element Plus（后台管理）

   * 配置路由（Vue Router）

   * 配置状态管理（Pinia）

   * 配置Axios请求拦截器

   * 配置环境变量（开发/生产）

### 第二阶段：数据库设计与创建

1. **数据库表结构设计**

   * 创建数据库 `skill_chain`

   * 设计并创建以下核心表：

     * 用户体系：`user`, `wallet`, `decoration`

     * 技能排期：`skill`, `schedule`, `skill_category`

     * 订单交易：`order`, `transaction_log`

     * 社交任务：`task`, `user_task`, `review`

     * 系统管理：`admin`, `appeal`, `system_config`

### 第三阶段：核心功能模块实现

1. **用户认证与权限模块**

   * 用户注册/登录（JWT认证）

   * 角色权限控制（普通用户/工作者/管理员）

   * 实名认证功能

   * 密码加密与验证

2. **个人中心模块**

   * 用户资料管理

   * 虚拟装饰系统（头像框、头衔）

   * 钱包系统（CNY挂钩币 + 积分双币）

   * 每日签到（Redis BitMap）

3. **技能与排期模块**

   * 技能发布与管理

   * 排期配置（时间槽Time Slot机制）

   * 技能搜索与分类

   * 多媒体上传（图片/视频）

4. **订单与支付模块**

   * 预约下单流程

   * 支付系统（模拟支付接口）

   * 订单状态管理（状态机模式）

   * 退款与仲裁

5. **游戏化任务系统**

   * 任务字典配置

   * AOP切面实现任务触发

   * 积分奖励发放

   * 积分商城

6. **后台管理模块**

   * 用户管理与审核

   * 技能内容审核

   * 订单监控与仲裁

   * 财务报表与提现审批

   * 系统配置管理

### 第四阶段：优化与测试

1. **性能优化**

   * Redis缓存优化

   * 数据库索引优化

   * 前端懒加载与代码分割

2. **测试与文档**

   * 单元测试

   * 接口测试

   * 编写API文档

   * 准备论文素材
